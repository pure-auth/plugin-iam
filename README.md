# @pure-auth/plugin-iam

Role, permission, resource, and API-key IAM plugin for Pure Auth.

## Repository

This package is intended to live in its own repository:

- GitHub: `pure-auth/plugin-iam`
- npm: `@pure-auth/plugin-iam`
- Peer dependencies: `pure-auth`, `@pure-auth/plugin-organization`

## Development

```bash
bun install
bunx tsc --noEmit
```
